# Portfolio Website

This project contains a responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Structure

- `frontend/`
  - `index.html` - Main portfolio page
  - `style.css` - Responsive styling
  - `script.js` - Interactive features and smooth scrolling
  - `assets/` - Images and other assets

## Features

- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Modern UI with smooth animations
- ⚡ Smooth scroll navigation
- 🎯 Sections for Home, About, Skills, Projects, Education, and Contact
- ✨ Interactive elements with hover effects
- 🎭 Timeline for education history

## Run locally

Simply open `frontend/index.html` in any modern browser:

```bash
# On Windows
start frontend/index.html

# On macOS
open frontend/index.html

# On Linux
xdg-open frontend/index.html
```

Or you can use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Then open http://localhost:8000 in your browser
```

## Customization

- Update contact information in the contact section
- Add your own images to the `assets/` folder
- Modify the color scheme in `style.css` (look for `:root` CSS variables)
- Update project information in the projects section
