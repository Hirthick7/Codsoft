# CODSOFT Projects

A multi-project repository containing frontend and backend apps.

## Projects

- `portfolio-website/`
  - A personal portfolio website built with HTML, CSS, JavaScript, and static assets.
  - Showcases skills, projects, education, and contact information
  
- `calculator-app/`
  - A simple calculator application with a Python Flask backend
  - HTML/CSS/JavaScript frontend with API integration

## Quick Navigation

### Portfolio Website
- Location: `portfolio-website/frontend/`
- How to run: Open `index.html` in a browser
- Technologies: HTML, CSS, JavaScript

### Calculator App
- Location: `calculator-app/`
- Frontend: `frontend/` (HTML, CSS, JavaScript)
- Backend: `backend/` (Python Flask)
- How to run:
  1. Install dependencies: `pip install -r backend/requirements.txt`
  2. Run backend: `python backend/server.py`
  3. Open browser: `http://127.0.0.1:5000`
